在使用Hexo搭建博客之前，先来了解一下什么是Hexo？

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

Ⅰ. 安装node:

1. 下载：

# wget https://nodejs.org/dist/v12.10.0/node-v12.10.0-linux-x64.tar.xz

2. 解压：

# tar xf node-v12.10.0-linux-x64.tar.xz

3. 解压文件的 bin 目录底下包含了 node、npm 等命令，我们可以使用 ln 命令来设置软连接：

# ln -s /usr/local/blog/node/bin/npm /usr/local/bin/
# ln -s /usr/local/blog/node/bin/node /usr/local/bin/
路径需根据自己文件的路径设置

4. 查看node版本

# node --version

Ⅱ. git安装

1. 使用yum命令安装git-core

# yum install git-core

2. 查看git版本

# git version

Ⅲ. 安装hexo

1. 下载Hexo

# npm install hexo-cli -g

2. 查看Hexo版本

hexo version

3. 设置环境

# ln -s /usr/local/blog/node/bin/hexo /usr/local/bin/

4. 生成站点目录

# hexo init <folder>
# cd <folder>
# npm install
在第一步”hexo init “完成以后可能会出现报错，无须在意，按顺序执行完所有步骤以后即可。

Ⅳ. 开放4000端口（未必每个人都能用上，可按需跳过该步骤）

1. 查看防火墙状态

# firewall-cmd --state
　如果返回not running，代表防火墙没有启动
　　检查防火墙启动状态：
　　　# systemctl status firewalld
　　　　dead状态，即防火墙未开启
　　　开启防火墙：
　　　　# systemctl start firewalld
　　　关闭防火墙：
　　　　# systemctl stop firewalld

2. 添加端口

# firewall-cmd --zone=public --add-port=4000/tcp --permanent

3. 重启防火墙

# firewall-cmd --reload

4. 查看端口状态

# firewall-cmd --zone=public --query-port=4000/tcp

Ⅴ. 生成网站页面

# hexo generator
简写：
# hexo g

会生成public文件，该文件就是生成的网站
每一次写作完成以后都要执行该指令

Ⅵ. 启动hexo

# hexo server
简写：
# hexo s

Ⅶ. 打开博客

http://localhost:4000
